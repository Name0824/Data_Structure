# 깃허브 요약
- Fork
> Fork 는 자신의 저장소로 다른 사람의 프로젝트를 통째로 복사 하는 행위

- Clone
> 원격 저장소 | 파일 복사 행위

- Branch
> 개발용 코드 버전 생성
> 추후 master에 합치는 용도로 사용

- Add, Commit, Push
> 추가, 갱신, 업로드

- Pull Request
> 자신이 수정한 파일을 다른 사람 프로젝트에 반영 요청하는 것
> 상대방이 마음에 들면 그것을 merge 함
